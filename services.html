What I Do

My Services

Here is a detailed breakdown of everything I can help you build, ship, and document.

Frontend Web Development

I design and build responsive, fast, and modern websites tailored to your brand. Whether you need a landing page, a portfolio, or a full business website, I use HTML, CSS, Bootstrap 5, and Next.js to deliver clean, mobile-first experiences.

What's included: Responsive design, cross-browser compatibility, Bootstrap component customization, performance optimization, and deployment.

HTML & CSS Bootstrap 5 JavaScript Next.js

Full-Stack Web Applications

I build end-to-end web applications that go beyond static pages — handling user data, authentication, contact forms, and database-backed features. I use Next.js on the frontend and backend, with PostgreSQL and Prisma ORM for the data layer.

What's included: Frontend UI, backend API routes, database schema design, form handling, email notifications, and deployment.

Next.js TypeScript PostgreSQL Prisma ORM

Mobile App Development

I build cross-platform mobile applications for Android and iOS using React Native and Expo. From state management to API integration and navigation, I can take your mobile idea from concept to a working app.

What's included: Screen design, navigation setup, state management with Zustand, data fetching with TanStack Query, and API connectivity.

React Native Expo Zustand TanStack Query

Database Design

I design clean, scalable PostgreSQL database schemas tailored to your application's data needs. I also handle JDBC integration for Java applications and Prisma ORM setup for Next.js projects, ensuring your data layer is solid from day one.

What's included: Schema design, table relationships, indexing, JDBC/Prisma integration, and query optimization basics.

PostgreSQL Prisma ORM JDBC

API Integration

I connect your application to the third-party services it needs — email providers, payment gateways, data sources, and more. I've worked with SendGrid for transactional email, Axios for HTTP requests, and TanStack Query for data synchronization.

What's included: API setup, authentication handling, error handling, and data mapping to your UI.

Axios SendGrid REST APIs TanStack Query

Tech Content Creation

I create engaging, accurate tech content for businesses and personal brands — covering software engineering, data science, and cybersecurity. I run a tech-focused Instagram page and understand how to communicate complex topics clearly to different audiences.

What's included: Blog posts, LinkedIn articles, Instagram carousel posts, explainer threads, and technical write-ups.

Blog Writing Social Media Technical Writing

Desktop App Development

I build Java desktop applications with JavaFX for businesses that need offline-capable, database-connected GUI tools. From form-driven data entry systems to reporting dashboards, I can deliver a polished desktop product backed by PostgreSQL.

What's included: JavaFX UI design, event handling, JDBC database integration, and PostgreSQL backend.

Java JavaFX PostgreSQL JDBC

IT Project Documentation

I help startups and small teams put structure around their projects — writing clear requirements, project plans, and workflow documentation using SDLC and PMBOK frameworks. Good documentation saves time, reduces miscommunication, and keeps development on track.

What's included: Project scope documents, requirements specifications, SDLC model selection, workflow diagrams, and handover documentation.

SDLC PMBOK Requirements Writing Workflow Design
